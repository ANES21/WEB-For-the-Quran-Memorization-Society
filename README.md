# 🕌 منظومة إدارة جمعية تحفيظ القرآن الكريم (مكتب ملاكو)

منظومة برمجية متكاملة ومحمية لإدارة الحلقات القرآنية والأفواج التربوية، متابعة الأوراد والترقيات اليومية للطلاب، وتقييد الاشتراكات المالية والخزينة السحابية لجمعية المعالي للعلوم والتربية.

---

## 🚀 تقنيات المشروع (Tech Stack)

* **الواجهة الأمامية (Frontend):** HTML5, Tailwind CSS, JavaScript (Async/Fetch API).
* **الخلفية (Backend):** Node.js, Express.js.
* **قاعدة البيانات (Database & Auth):** Supabase (PostgreSQL) مدمج بنظام الحماية والتوكنز السحابي.

---
